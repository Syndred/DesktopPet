# 隐私合规清单（S-04）

文档版本：v1.0  
更新时间：2026-03-04

## 1. 数据采集

1. 仅采集行为统计，不采输入正文。
2. 默认不采精确 GPS，需独立授权后启用。
3. 禁止采集以下字段：
   - 邮箱
   - 手机号
   - 身份证号
   - 聊天正文

## 2. 存储与保留

1. 遥测数据默认保留不超过 180 天。
2. 日志导出必须脱敏（邮箱、手机号、Token）。
3. 高敏感字段不得进入分析仓。

## 3. 权限与告知

1. 定位、通知、后台运行权限需按场景弹窗说明。
2. 隐私政策应明确说明采集项、目的、保留时长。
3. 用户可在设置中关闭非必要采集。

## 4. 审计流程

1. 每次版本发布前执行隐私审计。
2. 使用 `auditPrivacyCompliance` 对关键埋点 payload 做白名单校验。
3. 审计不通过禁止发布。
